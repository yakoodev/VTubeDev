# VTube

Repo structure:
- unity/ — Unity URP project
- app/ — local server/orchestrator
- shared/ tools/ examples/ docs/

Notes:
- Do not commit secrets (.env).
- Keep Unity .meta files intact.

## Quick start (current state)
- Open the Unity project in `unity/`.
- Open scene `unity/Assets/Scenes/Sandbox.unity`.
- Enter Play Mode to verify the project loads.

## App/server
The `app/` folder is a placeholder at the moment. No runnable server is defined yet in this repo.
