# F2_2 — Плагины: порядок, lifecycle, безопасные вызовы

**Status:** todo

## Цель

Добавить порядок плагинов и стабильный lifecycle (Init/OnTick/Dispose).

## Контекст / ссылки

- `docs/vtube-mvp.md` (Plugins lifecycle)
- Зависит от F2_1

## Объём

**Включено:**

- Настраиваемый порядок загрузки
- OnTick/OnEvent вызовы
- Dispose при остановке

**Не включено:**

- Загрузка по сетке доверия

## Шаги реализации

1. Ввести порядок (по конфигу или алфавиту).
2. Реализовать lifecycle callbacks.
3. Логировать ошибки и продолжать работу.

## Критерии приёмки

- [ ] Плагины вызываются в заданном порядке.
- [ ] Ошибка одного плагина не ломает остальные.

## Как проверить

- Два тестовых плагина с явным порядком.

## Артефакты

- Плагин lifecycle
- Конфиг порядка
