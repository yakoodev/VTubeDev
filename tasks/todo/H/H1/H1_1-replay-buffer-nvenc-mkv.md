# H1_1 — Replay buffer + NVENC H.264 + MKV

**Status:** todo

## Цель

Сделать replay буфер с записью последних N секунд и сохранением в MKV (NVENC H.264).

## Контекст / ссылки

- `docs/vtube-mvp.md` (Replay)

## Объём

**Включено:**

- Ring buffer N секунд
- Триггер сохранения
- NVENC H.264 → MKV

**Не включено:**

- Мультикамерная запись

## Шаги реализации

1. Ввести буфер кадров на камеру.
2. Реализовать триггер сохранения.
3. Подключить энкодер NVENC H.264.

## Критерии приёмки

- [ ] Сохраняется MKV с последними N секундами.
- [ ] Буфер удерживает заданную длину.

## Как проверить

- Включить bufferize, триггернуть сохранение, открыть MKV.

## Артефакты

- Replay buffer
- Encoder pipeline
