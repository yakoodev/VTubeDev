# C4_1 — Node editor read-only визуализация

**Status:** todo

## Цель

Сделать read-only визуализацию графов/конфигов (node editor), как в MVP.

## Контекст / ссылки

- `docs/vtube-mvp.md` (Node editor read-only)

## Объём

**Включено:**

- Отображение узлов и связей
- Read-only режим

**Не включено:**

- Редактирование графа

## Шаги реализации

1. Определить минимальную модель node/edge.
2. Отрисовать граф в UI (Unity UI или Web).
3. Подключить к конфигам (data/control graph).

## Критерии приёмки

- [ ] Граф виден и соответствует конфигу.
- [ ] Режим только чтение.

## Как проверить

- Открыть граф и сверить количество нод/связей с конфигом.
