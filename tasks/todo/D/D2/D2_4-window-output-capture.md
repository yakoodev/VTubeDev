# D2_4 — Window output: локальное окно для Window Capture

**Status:** todo

## Цель

Сделать локальное окно вывода для OBS Window Capture как fallback.

## Контекст / ссылки

- `docs/vtube-mvp.md` (Window Capture fallback)

## Объём

**Включено:**

- Окно вывода для выбранного formatId/profile
- Переключение формата

**Не включено:**

- Множественные окна одновременно

## Шаги реализации

1. Создать окно вывода в Unity.
2. Привязать к выбранному formatId/profile.
3. Обновлять кадр в окне.

## Критерии приёмки

- [ ] Окно показывает актуальный кадр.

## Как проверить

- Открыть окно и проверить в OBS Window Capture.

## Артефакты

- Window output
