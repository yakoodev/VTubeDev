# G1_0 — Base Debug UI: панель для input sources и камер

**Status:** todo


## Цель

Добавить минимальную runtime Debug UI панель, чтобы задачи по выбору источника и камеры можно было проверять без отдельного UI-фреймворка.


## Контекст / ссылки

G1_3 и G1_4 требуют dropdown/toggle для выбора источника и устройства камеры. Сейчас базовой панели нет.


## Объём

**Включено:**

- Минимальная runtime панель (Unity UI) с возможностью включать/выключать
- Dropdown списка доступных webcam устройств (пустой state ок)
- Toggle/кнопки для выбора VideoSim/Webcam режима
- Отображение активного режима и статуса источника (IsReady/Resolution)


**Не включено:**

- Полноценный UI стиль/темизация


## Шаги реализации

1. Создать простую Debug UI панель (Canvas) в Sandbox сцене или как префаб.
2. Добавить dropdown для камеры (данные подставляет IFrameSource/manager).
3. Добавить toggle/кнопки для выбора источника.
4. Добавить текст статуса (активный источник, размер кадра, IsReady).


## Критерии приёмки

- [ ] Панель включается/выключается (например горячей клавишей или кнопкой).
- [ ] Можно выбрать источник и увидеть текущий статус.
- [ ] При отсутствии устройств UI не ломается и показывает пустой список.


## Как проверить

- Открыть Sandbox, показать панель, переключить VideoSim/Webcam.
- Если камеры нет — убедиться, что UI не падает и отображает статус.


## Артефакты

- Debug UI панель
- Скриншот панели (по задаче, если требуется)
