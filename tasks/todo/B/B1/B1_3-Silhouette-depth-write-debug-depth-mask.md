# B1_3 — Silhouette depth write + debug depth mask

**Status:** todo


## Цель

Реализовать depth-write силуэта, чтобы перекрытия 2D/3D были предсказуемыми (как в холсте), и добавить debug маску.


## Объём

**Включено:**

- Depth pass
- Order control
- Debug depth view


**Не включено:**

- Multi-camera сложные кейсы




## Шаги реализации

1. Добавить pass, который пишет depth из выбранной маски/геометрии.
2. Использовать depth при композите, чтобы 3D мог перекрывать 2D корректно.
3. Добавить debug режим показать depth mask.


## Критерии приёмки

- [ ] Есть сцена, где 3D объект частично перекрывает 2D — результат корректный.
- [ ] Debug depth view доступен переключателем.


## Как проверить

- Скрин: normal view + depth view.


## Артефакты

- Depth pass
- Debug toggle
- Тестовая сцена
