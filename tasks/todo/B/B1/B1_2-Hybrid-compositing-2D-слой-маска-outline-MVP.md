# B1_2 — Hybrid compositing: 2D слой + маска + outline (MVP)

**Status:** todo


## Цель

Добавить гибридный слой: 2D (спрайт/квад) поверх 3D с маской и простым outline — чтобы дальше на нём строить аватар/оверлеи.


## Объём

**Включено:**

- 2D layer
- Alpha mask
- Outline параметр


**Не включено:**

- Сложные эффекты/шейдер граф




## Шаги реализации

1. Сделать компонент HybridLayer2D (texture/material).
2. Реализовать blending с BaseRT.
3. Outline: простой материал с настраиваемой толщиной (хотя бы float).


## Критерии приёмки

- [ ] 2D слой виден поверх 3D.
- [ ] Outline заметно меняется при настройке.


## Как проверить

- Добавить тестовый спрайт и сделать скриншот до/после изменения толщины.


## Артефакты

- HybridLayer2D
- Материал/шейдер
- Тестовая сцена/префаб
