# Пример аватара: avatar-sim.mp4

В качестве эталонного видео для эмуляции веб-камеры лежит файл `tasks/avatar/avatar-sim.mp4`.

- Контейнер: ISO-MP4 (`mp4`).
- Кодек видео: `avc1` (H.264).
- Разрешение: 1920×1080 (уточнено из заголовка `tkhd`).
- Поток рассчитан на ~30 fps, но если нужна точная частота, посмотри параметры источника или пересоздай видео.

Чтобы перепроверить можно запустить любой MP4-инспектор (например, `ffprobe`, `MediaInfo`, `mp4info`) на этом файле и сравнить вывод с этими значениями. Если появится новый эталон, заменяй файл и обновляй заметку.
